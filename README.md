CHMpad is a full-fledged portable code editor contained within a single Windows Help file (.chm). 
Features: 
**It is just a few kilobytes in size, features a dark theme and the built-in Consolas font, bypasses the IE sandbox to open and save files, and includes automatic saving to localStorage on every keystroke!**

# CHMpad
CHMpad — это полноценный портативный текстовый редактор для кода, запертый внутри ОДНОГО файла справки Windows (.chm).
Фичи: 
**Весит пару килобайт, имеет темную тему, встроенный шрифт Consolas, обходит песочницу IE для открытия/сохранения файлов и обладает встроенным автосохранением в localStorage на каждое нажатие клавиши!**
